El ejercicio plantea como base la creación de estructuras de datos ya conocidas (listas y mapas), y especificadas en los requerimientos del ejercicio. Para cada plataforma, se usa una lista de mapas, las cuales son adecuadas para agregar CLAVE-VALOR, y registrar varias plataformas si así se desea, donde entra en juego la lista.

Una vez creadas con la información requerida, se procede a trabajar con ellas. Se crea una función, un mapa que recibe como parámetro la lista de mapas.

Dentro de la función se encuentra una lista de mapa vacia, donde se almacenará la información que se esta solicitando, a través de ciclo FOR y condicionales con los requerimientos que se estan buscando filtrar. Si el if se cumple, se almacena la información en la lista de mapa vacia.

En la función main, mediante un forEach, se recorre la función, quedando de manera ordenada. Es decir, analizarPlataforma() devuelve un Map, dentro del Map hay una List, forEach debe ir sobre la List.
