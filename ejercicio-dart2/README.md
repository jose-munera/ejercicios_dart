Con el paso de la tecnología, los equipos móviles se quedan obsoletos debido a la innovación en el software. Se busca que ciertas versiones de Android e iOS no queden por fuera aún.

Se tómo la decisión de usar una Lista cuyos elementos son mapás, de manera cada elemento tenga clave y valor, lo que hace que se sepa de qué se esta hablando.

Se crea una función que recibe una lista de mapas y retorna un mapa de estrategia de retrocompatibilidad, donde se reciben las plataformas y luego se evaluan mediante ciclo for y condicionales.
