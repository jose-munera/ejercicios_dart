Este ejercicio modela el dilema que conlleva entre elegir la mejor opción de acuerdo al gasto promedio del usuario

La solución propuesta permite identificar la plataforma que genera mayor rentabilidad de acuerdo a los datos.

Desde una perspectiva empresarial, este enfoque facilita la toma de decisiones de acuerdo a los datos en las fuentes de acuerdo con reportes de 2025, es decir, información confiable.

Se tomó las decisiones técnicas debido a que los Mapas permiten escribir una clave y un valor, que es justo lo que requiere el ejercicio, ya a partir de allí, lo que se hacen son consultas como si se tratara de una base de datos.
